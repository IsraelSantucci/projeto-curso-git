Comandos do git

  • git init - iniciar um repositório
  • git add . - adiciona todos os arquivos para o commit
  • git status - mostra o estado do arquivos. adicionado ou nao para poder ser commitado
  • git commit -m 'mensagem do commit' - faz o commit do arquivo
  • git remote add origin git@github.com:IsraelSantucci/projeto-curso-git.git - fala para o git onde vai ser nossa origem de repositorio remoto
  • git push -u origin master - manda para o repositorio as nossas alterações
  • git clone git@github.com:IsraelSantucci/projeto-curso-git.git - faz um clone do reposito remoto e o coloca na sua maquina
  • git branch - mostra as branchs do projeto.
  • git checkout "nomeDaBranch" - vai para a branch desejada.
  • git branch "novaBranch" - cria uma nova branch.
  • git checkout -b "novaBranch" - cria uma nova branch e ja troca de branch para poder usa-la
  • git branch -D "nomeBranch" - deleta um branch
  • git push origin "nomeBranch" - envia a branch para o repositório remoto.
